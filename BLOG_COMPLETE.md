## 💡 **Cum Să Adaugi Articole Noi**

1. **Creează fișier nou** în `src/pages/blog/slug-articol.astro`

2. **Folosește template**:
```astro
---
import Layout from '../../layouts/Layout.astro';
import BlogPost from '../../components/Blog/BlogPost.astro';
import BlogPostSchema from '../../components/SEO/BlogPostSchema.astro';
import CTASection from '../../components/UI/CTASection.astro';

const postData = {
  title: 'Titlu Articol',
  excerpt: 'Descriere scurtă...',
  date: 'Data',
  category: 'Categorie',
  image: 'URL imagine',
  readTime: 'X min citire',
  slug: 'slug-articol'
};
---

<Layout title={`${postData.title} | Blog InuWood`} description={postData.excerpt}>
  <BlogPost {...postData}>
    <!-- Conținut articol aici -->
    <h2>Heading 2</h2>
    <p>Paragraf...</p>
  </BlogPost>

  <CTASection 
    heading="CTA Heading"
    description="..."
    ctaText="Buton"
    ctaLink="/contact"
  />

  <BlogPostSchema {...postData} description={postData.excerpt} />
</Layout>
```

3. **Adaugă în lista de blog** (`src/pages/blog.astro` - array `blogPosts`)

4. **Build și testează**: `npm run build`
