## Descriere

Acest set cu margine naturală live edge celebrează forma originală a copacului. Fiecare piesă păstrează marginea naturală a lemnului de frasin, făcând fiecare set unic și irepetabil.

### Caracteristici

- **Lemn**: Frasin masiv, margine live edge păstrată
- **Set**: Platou de servire + tocător
- **Finisaj**: Ulei de in alimentar și ceară de albine
- **Unicitate**: Fiecare set este diferit datorită marginii naturale

### Proces de Realizare

Marginea live edge este păstrată și curțțită manual de coajă și impurități. Suprafața este netezită până la un finisaj neted și sigur pentru alimente, în timp ce marginea exterioară păstrează textura naturală a copacului.

### Îngrijire

Spălați manual. Nu lăsați în apă. Marginea live edge necesită o uscare imediată după contactul cu lichide. Aplicați ulei de in periodic.
